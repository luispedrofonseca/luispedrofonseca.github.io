ProCamera2D · Plugins · Unity

One of the main goals of ProCamera2D’s new extension - Rooms - was to make it as user-friendly as possible. Each room is basically a rectangle with some extra data to allow custom transitions. To make it easy to edit the rooms the user should be able to interact with them on the scene. However, Unity does not have a built-in rectangle (Rect) editor tool. Gladly, Unity is very flexible and easy to extend, so I created a simple one.
